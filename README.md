# Käyttöönotto- ja huoltotuen demo

Interaktiivinen staattinen demo autonpesulaitteiden käyttöönoton dokumentoinnista, huollon ohjatusta vianhausta ja hallitusta eskaloinnista.

## Mitä demo näyttää

- vakioitu käyttöönottochecklist ja luovutusvalmius
- huollon itsepalvelupolut ennen asiantuntijaeskalointia
- haettava dokumenttikirjasto koko laitteen elinkaarelle

## Julkaisu GitHub Pagesiin

1. Luo GitHubiin uusi repository.
2. Lisää tämän kansion tiedostot repositoryyn.
3. Avaa GitHubissa `Settings` -> `Pages`.
4. Valitse lähteeksi `Deploy from a branch`.
5. Valitse branchiksi `main` ja kansioksi `/ (root)`.
6. Tallenna. GitHub luo julkisen linkin, joka avaa `index.html`-sivun automaattisesti.

## Tiedostot

- `index.html` sisältää demon rakenteen
- `demo.css` sisältää ulkoasun
- `demo.js` sisältää interaktiot
